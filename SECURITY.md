# Security policy

This repository contains **data-driven mod scripts** and documentation for **Crusader Kings III**. It does not ship a standalone executable, server, or network service as part of the mod source tree.

## Supported versions

Security-sensitive fixes (for example, unsafe instructions that could encourage harmful practices) are considered only for the **current default branch** and the **latest tagged release** intended for players.

## Reporting a vulnerability

If you believe you have found a **security vulnerability** related to this repository (for example, compromised automation in CI, malicious links in documentation, or sensitive credentials accidentally committed):

1. **Do not** open a public issue with exploit details.
2. Instead, use **GitHub private vulnerability reporting** if enabled for this repository (**Security → Report a vulnerability**), or contact the maintainers through a private channel they publish on the repository main page.

If private reporting is not available, open a generic issue asking maintainers for a contact method, without including sensitive exploit material.

## Out of scope

- **Game engine** or **Paradox launcher** vulnerabilities — report those to [Paradox Interactive](https://www.paradoxinteractive.com/) through their official channels.
- **Gameplay exploits** inside CK3 (for example, war score manipulation) — use [GitHub Issues](https://github.com/TitaniumKnight1/Better-Automated-Armies/issues) for normal bug reports unless they involve real-world harm or leaked secrets.
